#Moduralizacion 
Calculadora básica — Recibe dos números y una operación (suma, resta, multiplicación, división). Cada operación tiene su propio método y se valida que no haya división entre cero.
Validación de contraseña — Pide la contraseña en un ciclo do-while y no permite avanzar hasta que el usuario ingrese "1234" correctamente.
Números primos — Recibe un número entero y mediante la función esPrimo() determina si es primo o no, usando raíz cuadrada para optimizar la búsqueda.
Suma de pares — Acumula en un while todos los números pares que el usuario ingrese, deteniéndose cuando escribe 0.
Conversión de temperatura — Permite convertir entre Celsius y Fahrenheit en ambas direcciones, con un método dedicado para cada conversión.
Contador de vocales — Recibe cualquier texto y cuenta sus vocales, incluyendo mayúsculas y letras con tilde, usando la función contarVocales().
Factorial — Calcula el factorial de un número no negativo con un ciclo for, usando long para soportar resultados grandes.
Juego de adivinanza — Genera un número aleatorio entre 1 y 100 y guía al usuario con pistas de mayor o menor en un do-while hasta que adivine.
Paso por referencia — Simula el intercambio de dos variables usando una clase interna Contenedor, ya que Java no tiene paso por referencia nativo para primitivos.
Tabla de multiplicar — Recibe un número y muestra su tabla del 1 al 10 formateada, generada dentro del método mostrarTablaMultiplicar().
